## reference type
```
var object1 = { value: 10};
var object2 = object1;
var object3 = { value: 10};
```
- object2 references the address of object1, while objects create a new space.
```
object1.value = 15;
object2.value;
```
=> 15
```
object3.value;
```
=> 10.
