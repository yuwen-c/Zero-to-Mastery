## beginning
- prepare a express server:
  - install package.json:
  ```
  $ npm init -y
  ```
  - install express
  ```
  $ npm install express
  ```
  - install nodemon
  ```
  $ npm install nodemon --save-dev
  ```
  - edit script
  ```
  "start" : "nodemon script.js" // depend on file name
  ```
