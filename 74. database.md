## postgreSQL
### install
```
$ brew install postgresql
```

### use
- start
```
$ brew services start postgresql
```
- stop
```
$ brew services stop postgresql
```
- create a database
```
$ createdb 'DatabaseName'
```
- connect to it
```
$ psql 'DatabaseName'
```
