## a portal component
- written in a class
- constructor:
```
// create a new div element 
this.el = document.createElement("div")
```
- life cycle functions:
  - componentDidMount:
  ```
  appendChild(this.el)
  ```
  - componentUnMount
  ```
  removeChild(this.el)
  ```
- use createPortal:
```
return createPortal(
<div></div>, this.el
)

// content, container
```
