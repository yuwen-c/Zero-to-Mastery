## array

- function can be part of array.
```
var array = [function apple(){
console.log("apple");}]

array[0]();
```
- mixed array also exists, but NOT RECOMMENDED.
```
var mixedList = [
"yo", 8, true, undefined, function apple(){
console.log("apple");}]
```
- we can also make arrays in an array.
```
var list = [
["tiger", "pi", "ship", "An Li"]
]
```
