## express- server module
```
$ npm install express
```

## 3 steps to build a server

- import module
```
const express = require("express")
```

- create one
```
const app = express();
```

- set port
```
app.listen(3000)
```

## get method
```
app.get("/", (req, res) => {
    const user = {
        name: 'judy',
        age: 40
    }
    res.send(user)
})
```
**** express does JSON parsing for us.
