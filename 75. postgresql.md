## postgreSQL 

### basic commands
- start
```
$ brew services start postgresql
```
- stop
```
$ brew services stop postgresql
```
- create a database
```
$ createdb "DatabaseName"
```
- connect GUI to posgresql
```
$ psql "DatabaseName"
```

#### after connecting to posgresql...

- check database
```
$ \d
```
- drop connecting
```
$ \q
```
